# UICreator
Application to design UI and get corresponding code. Makes development easy.

## Setup Locally

### Prerequisites
 - NPM (Version 6.14.9)
 - NODE (Version 14.15.3)

### Installation
  #### Clone repo
      git clone https://github.com/himanshuvarandani/UICreator.git
      cd UICreator
  #### Install Requirements
      npm install
  #### Start Application
      npm start
